# Studies in C

Personal repository to track my learning journey in C programming language,
as part of my Software Engineering degree.

## About

I started learning C in March 2026 as the foundation for systems programming,
Linux kernel development and Rust. This repository contains all exercises,
class assignments and personal projects from my studies.

## Structure
```
studies-c/
├── aula01/               # Lesson 01 — Variables and types
│   ├── hello.c           # First program — Hello World
│   ├── variaveis.c       # Variables: char, int, float
│   └── operadores.c      # Arithmetic operators
│
└── aula02/               # Lesson 02 — Input, output and math
    ├── exercicio1.c      # Variables with user input
    ├── exercicio2.c      # Grade average calculator
    ├── problema1.c       # Circle area calculator
    └── problema2.c       # Grade average calculator
```
